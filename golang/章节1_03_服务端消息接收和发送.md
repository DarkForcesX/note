### 一、服务端接受数据并返回数据

***

* 在服务端添加循环，不停接受客户端发送来的数据
* 服务端代码

```go
package main

import (
	"fmt"
	"net"
)

func main() {
	//1、创建服务器地址
	addr, _ := net.ResolveTCPAddr("tcp4", "localhost:8899")
	//2、创建监听器
	lis, _ := net.ListenTCP("tcp4", addr)
	fmt.Println("v2服务器已经启动")
	for {
		//3、通过监听器获取客户端传递过来的数据
		//阻塞式
		conn, _ := lis.Accept()
		//4、转换数据
		b := make([]byte, 1024)
		n, _ := conn.Read(b)
		fmt.Println("获取数据为：", string(b[:n]))
		//5、发送消息
		conn.Write(append([]byte("server"), b[:n]...))
		//关闭连接
		conn.Close()
	}
}
```

* 客户端代码

```go
package main

import (
	"fmt"
	"net"
	"strconv"
	"time"
)

func main() {
	//1、创建服务器端地址
	addr, _ := net.ResolveTCPAddr("tcp4", "localhost:8899")
	for i := 0; i < 5; i++ {
		//2、创建连接
		conn, _ := net.DialTCP("tcp", nil, addr)
		//3、发送数据
		conn.Write([]byte("客户端发送端数据" + strconv.Itoa(i)))
		//4、读取服务端发回的消息
		b := make([]byte, 1024)
		count, _ := conn.Read(b)
		fmt.Println("服务端发送回来消息：", string(b[:count]))
		//5、关闭连接
		conn.Close()
		time.Sleep(1e9)
	}
}
```

### 二、并发访问

***

* 利用goroutine完成并发访问
* 修改server.go，在里面添加goroutine

```go
package main

import (
	"fmt"
	"net"
	"time"
)

func main() {
	//1、创建服务器地址
	addr, _ := net.ResolveTCPAddr("tcp4", "localhost:8899")
	//2、创建监听器
	lis, _ := net.ListenTCP("tcp4", addr)
	fmt.Println("v2服务器已经启动")
	for {
		//3、通过监听器获取客户端传递过来的数据
		//阻塞式
		conn, _ := lis.Accept()
		go func() {
			//4、转换数据
			b := make([]byte, 1024)
			n, _ := conn.Read(b)
			fmt.Println("获取数据为：", string(b[:n]))
			//5、发送消息
			//time.Sleep(1e9)
			conn.Write(append([]byte("server"), b[:n]...))
			//关闭连接
			conn.Close()
		}()
	}
}
```

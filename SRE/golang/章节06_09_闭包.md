#### 一、闭包概述

***

* 闭包解决局部变量不能被外部访问的一种解决方案
* 把函数当作返回值的一种应用

#### 二、代码演示

***

* 总体思想为：在函数内部定义局部变量，把另一个函数当作返回值，局部变量对于返回值函数就相当于全局变量，所以多次调用返回值函数局部变量的值跟随变化

```go
package main

import "fmt"

func main() {
	//res 是返回值函数
	res := test1()
	fmt.Println(res())
	fmt.Println(res())

	//重新调用,则i重新赋值一块新的内存空间
	res1 := test1()
	fmt.Println(res1())
	fmt.Println(res1())
}

//返回是 func() int
func test1() func() int {
	i := 1
	return func() int {
		fmt.Printf("%p \n", &i)
		i++
		return i
	}
}

```
